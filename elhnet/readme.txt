Readme for ELHnet Python 3.5 code.
Last updated: 9/16/2017	

To use:
(1) Open Spyder 2.3.9 (Ananconda)
(2) Run in command line: "%matplotlib qt"
(3) Run "elhnet.py" (main script)
(4) Select image to analyze in pop-up dialog box (may have to minimize current window to see dialog box)
(5) Click on center of 64x64 px ROI in image to classify 

All files should be in the same directory. 
If bugs or questions, email me at gsliu11@gmail.com. 

Files:
elhnet.py (main script)
fast_layersGL.py
im2col.py
mat.py
octcnn_weights_valacc0.99817232376.h5 (ELHnet model weights)
